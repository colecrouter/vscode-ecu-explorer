# Skills Directory

This directory contains companion agent skills for ECU Explorer workflows.

Conventions:

- one skill per subdirectory
- required file: `SKILL.md`
- keep skills small and task-focused
- prefer referencing MCP resources for long-form reference content
- do not assume the MCP server is connected unless the skill says how to verify it

Suggested structure:

```text
skills/
  ecu-log-analysis/
    SKILL.md
    examples/
  rom-tuning/
    SKILL.md
  definition-authoring/
    SKILL.md
```
